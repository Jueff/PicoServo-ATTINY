# PicoServo-ATTINY

(deutsche Version weiter unten)

Brief description
----------------
This project contains firmware for the Raspberry Pi Pico that emulates a 6× servo controller for MobaLedLib-based model railway setups. The firmware receives LED data, controls up to 6 servos (in two groups) and provides status feedback via addressable LEDs. Configurations (limits, speed, position) are stored in flash.

Key features
------------------
- Control of up to 6 servos (RP2040 / Pico)
- Persistent storage of servo parameters in Flash
- Interactive configuration modes (Min/Max, Speed)

Kurzbeschreibung
----------------
Dieses Projekt enthält die Firmware für den Raspberry Pi Pico, die ein 6×-Servo-Controller‑Verhalten für MobaLedLib‑basierte Modellbahn‑Setups emuliert. Die Firmware empfängt LED‑Daten, steuert bis zu 6 Servos (in zwei Gruppen) und bietet Statusfeedback über adressierbare LEDs. Konfigurationen (Limits, Speed, Position) werden in Flash gespeichert.

Wichtige Features
------------------
- Steuerung von bis zu 6 Servos (RP2040 / Pico)
- Persistente Speicherung der Servo‑Parameter im Flash
- Interaktive Konfigurationsmodi (Min/Max, Speed)
